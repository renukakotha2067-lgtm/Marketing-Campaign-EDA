# 📊 Marketing Campaign Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Marketing Campaign dataset to understand customer behavior, campaign effectiveness, purchasing patterns, and spending habits. The goal is to generate actionable business insights and recommend strategies to improve future marketing campaigns.

## 🎯 Objective

- Clean the dataset
- Perform Exploratory Data Analysis (EDA)
- Analyze customer demographics and purchasing behavior
- Evaluate marketing campaign performance
- Generate business insights
- Provide data-driven recommendations

## 🛠️ Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Dataset

Marketing Campaign Dataset containing customer demographics, purchase history, and campaign responses.

## 📈 Exploratory Data Analysis

The project includes:

- Data Cleaning
- Missing Value Treatment
- Duplicate Removal
- Feature Engineering
- Customer Age Analysis
- Income Distribution
- Spending Analysis
- Product Category Analysis
- Purchase Channel Analysis
- Campaign Response Analysis
- Marketing Funnel Visualization
- Correlation Heatmap
- Customer Segmentation

## 📊 Key Insights

- Higher-income customers tend to spend more.
- Wine products contribute the largest share of customer spending.
- Store purchases are higher than catalog and web purchases.
- Previous campaign responders are more likely to respond again.
- Customers with higher spending show stronger engagement.

## 💡 Business Recommendations

- Target high-income customers with premium campaigns.
- Retarget customers who responded to previous campaigns.
- Increase promotions for high-performing product categories.
- Improve website conversion through personalized offers.
- Focus marketing efforts on high-value customer segments.

## 🚀 Results

This analysis identifies customer segments with the highest purchasing potential and provides actionable recommendations to improve future marketing campaign effectiveness.
